Help Desk

Olá! Seja bem-vindo ao sistema de Help Desk desenvolvido por mim durante o curso Desenvolvedor Full Stack.

Este é um sistema para gerenciamento de chamados que simula um simples Help Desk, desenvolvido em Angular 12 e Spring Boot 2.

O sistema está sendo executado na nuvem, com o frontend hospedado na Vercel e o backend na Heroku.

O projeto tem como principal objetivo abordar os conceitos de Single Page Applications (SPAs), autenticação e autorização com Tokens JWT, e muito mais. O código fonte do projeto pode ser acessado clicando no link do Github no menu lateral.

Sinta-se à vontade para testar o sistema e deixar suas sugestões.
